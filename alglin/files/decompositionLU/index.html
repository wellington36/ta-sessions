Decomposição LU

Fatorização de uma matriz como um produto , onde a primeira matrix é triangular inferior (lower), enquanto a segunda é triangular superior (upper). A matriz é uma matriz de permutação que garante a existência dessa fatorização. 

import scipy.linalg

A = scipy.array([ [7, 3, -1, 2], [3, 8, 1, -4], [-1, 1, 4, -1], [2, -4, -1, 6] ])
print(A)

[[ 7  3 -1  2]
 [ 3  8  1 -4]
 [-1  1  4 -1]
 [ 2 -4 -1  6]]

P, L, U = scipy.linalg.lu(A)
print(P), print(L), print(U)
